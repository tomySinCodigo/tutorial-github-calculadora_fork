# Paso 2: Clonar y Abrir en VS Code

1. Copia la URL del repositorio (`Code > HTTPS`).
2. Abre la terminal y ejecuta:

```bash
git clone https://github.com/tu_usuario/calculadora-js.git
cd calculadora-js
```

3. Abre VS Code con:

```bash
code .
```
